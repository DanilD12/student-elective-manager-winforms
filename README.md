# Система учета студентов на факультативах (WinForms)

Десктопное приложение для автоматизации учета посещаемости и успеваемости студентов на факультативных занятиях. Реализовано на языке C# с использованием технологии Windows Forms.

## 🛠 Технологический стек
*   **Язык программирования:** C# (.NET Framework / .NET)
*   **Графический интерфейс:** Windows Forms (WinForms)
*   **Архитектура:** Разделение логики на уровни (хранение данных, модели объектов, интерфейс пользователя)
*   **Инструменты:** Visual Studio

## 🚀 Основной функционал
*   **Авторизация и регистрация:** Система управления учетными записями пользователей для защиты данных.
*   **Управление данными (CRUD):** Возможность добавления, редактирования, обновления и удаления записей о студентах.
*   **Фильтрация и поиск:** Гибкая система фильтрации списка студентов по предмету, оценке и ФИО.
*   **Сортировка:** Автоматическая сортировка данных по заданным параметрам.
*   **Генерация отчетов:** Функция формирования и сохранения индивидуальных заданий и общих отчетов в текстовый формат.

## 📁 Структура исходного кода
*   `LoginForm.cs` / `RegisterForm.cs` — модули аутентификации.
*   `MainForm.cs` — основное рабочее пространство со списками и фильтрами.
*   `Student.cs` / `User.cs` — модели данных (классы объектов).
*   `StudentStorage.cs` / `UserStorage.cs` — логика персистентного хранения и обработки данных (списков).
